\# CalculatorApp Testing Project



This project is a software testing project for a simple calculator application.



\## Project Description



The calculator application supports basic arithmetic operations and includes testing documentation and automated tests.



\## Features



\- Addition

\- Subtraction

\- Multiplication

\- Division

\- Input validation

\- Calculation history



\## Testing



The project includes both manual and automated testing.



Automated tests were created to verify calculator operations, input validation, and history functionality.

\- xUnit

\- ReportGenerator
